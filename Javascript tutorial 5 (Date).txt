
JS Math

Объект JavaScript Math позволяет выполнять математические задачи по номерам.

пример
Math.PI;            // returns 3.141592653589793


Math.round ()
Math.round (х) возвращает значение х, округленное до ближайшего целого числа его:

пример
Math.round(4.7);    // returns 5
Math.round(4.4);    // returns 4


Math.pow ()
Math.pow (х, у) возвращает значение х в степени у:

пример
Math.pow(8, 2);      // returns 64


Math.sqrt ()
Math.sqrt (х) возвращает квадратный корень из x:

пример
Math.sqrt(64);      // returns 8


Math.abs ()
Math.abs (х) возвращает абсолютное (положительное) значение х:

пример
Math.abs(-4.7);     // returns 4.7


Math.ceil ()
Math.ceil (х) возвращает значение х округляется до его ближайшего целого числа:

пример
Math.ceil(4.4);     // returns 5


Math.floor ()
Math.floor (х) возвращает значение х округляется вниз до ближайшего целого числа его:

пример
Math.floor(4.7);    // returns 4


Math.sin ()
Math.sin (х) возвращает синус (значение от -1 до 1) от угла х (заданный в радианах).

Если вы хотите использовать градусы вместо радианов, вы должны преобразовать градусы в радианы:

Угол в радианах = Угол в градусах х ПИ / 180.

пример
Math.sin(90 * Math.PI / 180);     // returns 1 (the sine of 90 degrees)


Math.cos ()
Math.cos (х) возвращает косинус (значение от -1 до 1) от угла х (заданный в радианах).

Если вы хотите использовать градусы вместо радианов, вы должны преобразовать градусы в радианы:

Угол в радианах = Угол в градусах х ПИ / 180.

пример
Math.cos(0 * Math.PI / 180);     // returns 1 (the cos of 0 degrees)


Math.min () и Math.max ()
Math.min () и Math.max () можно использовать, чтобы найти самое низкое или наибольшее значение в списке аргументов:

пример
Math.min(0, 150, 30, 20, -8, -200);  // returns -200
Math.max(0, 150, 30, 20, -8, -200);  // returns 150


Math.random ()
Math.random () возвращает случайное число в диапазоне от 0 (включительно) и 1 (не включается):

пример
Math.random();     // returns a random number

Math.random () используется с Math.floor () может быть использован для получения случайных чисел.

пример
Math.floor(Math.random() * 10);     // returns a number between 0 and 9
Math.floor(Math.random() * 11);      // returns a number between 0 and 10
Math.floor(Math.random() * 100);     // returns a number between 0 and 99
Math.floor(Math.random() * 101);     // returns a number between 0 and 100
Math.floor(Math.random() * 10) + 1;  // returns a number between 1 and 10
Math.floor(Math.random() * 100) + 1; // returns a number between 1 and 100


Математические свойства (Константы)
JavaScript предоставляет 8 математических констант, которые могут быть доступны с объектом Math:

пример
Math.E        // returns Euler's number
Math.PI       // returns PI
Math.SQRT2    // returns the square root of 2
Math.SQRT1_2  // returns the square root of 1/2
Math.LN2      // returns the natural logarithm of 2
Math.LN10     // returns the natural logarithm of 10
Math.LOG2E    // returns base 2 logarithm of E
Math.LOG10E   // returns base 10 logarithm of E


Математический конструктор
В отличие от других глобальных объектов, объект Math не имеет конструктора. Методы и свойства являются статическими.

Все методы и свойства (константы) могут быть использованы без создания объекта Math в первую очередь.

Методы Math объектов

abs(x)	Returns the absolute value of x
acos(x)	Returns the arccosine of x, in radians
asin(x)	Returns the arcsine of x, in radians

atan2(y, x)	Returns the arctangent of the quotient of its arguments
ceil(x)	Returns the value of x rounded up to its nearest integer
cos(x)	Returns the cosine of x (x is in radians)
exp(x)	Returns the value of Ex
floor(x)	Returns the value of x rounded down to its nearest integer
log(x)	Returns the natural logarithm (base E) of x
max(x, y, z, ..., n)	Returns the number with the highest value
min(x, y, z, ..., n)	Returns the number with the lowest value
pow(x, y)	Returns the value of x to the power of y
random()	Returns a random number between 0 and 1
round(x)	Returns the value of x rounded to its nearest integer
sin(x)	Returns the sine of x (x is in radians)
sqrt(x)	Returns the square root of x
tan(x)	Returns the tangent of an angle
